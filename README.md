# 🌐 Personal Portfolio Website

This is my **Personal Developer Portfolio** built using **React and Vite**.
It showcases my projects, skills, and contact information in a modern and responsive design.

## 🚀 Live Demo

https://thakurg1.github.io/PortFolio1/

## 🛠️ Technologies Used

* React
* Vite
* HTML5
* CSS3
* JavaScript
* Framer Motion (for animations)

## 📂 Featured Projects

### 🎮 Neon Tic Tac Toe

An interactive browser-based game with a glowing neon-themed UI where players compete in a classic tic-tac-toe game.

**Tech:** HTML, CSS, JavaScript

### 🔳 QR Code Generator

A web application that generates QR codes instantly by converting text or URLs into scannable QR codes.

**Tech:** HTML, CSS, JavaScript, QR API

### 🖼️ Auto Image Change

A simple web project that automatically changes images at a fixed time interval using JavaScript.

**Tech:** HTML, CSS, JavaScript

### 🌍 Hugo Static Website

A fast and lightweight static website created using the Hugo static site generator.

**Tech:** Hugo, HTML, CSS

## 📦 Installation

Clone the repository:

git clone https://github.com/yourusername/portfolio.git

Navigate to project folder:

cd portfolio

Install dependencies:

npm install

Run development server:

npm run dev
